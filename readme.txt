To run the program first start by doing python3 server.py to start the server and in a new terminal do python3 client.py.
Next you should see the menu in the client window. Follow the steps to withdraw deposit and check your balance.
Whenever you click an option in the client window the server should do the calculation and send back a response to be diplayed to
the client window. When you are ready to quit press 4 and you should quit the program.
